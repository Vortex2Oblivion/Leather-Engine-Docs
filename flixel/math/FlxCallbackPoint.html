class FlxCallbackPoint

package flixel.math

extends FlxBasePoint

import flixel.math.FlxPoint

@:directlyUsed@:build(hscript.macros.UsingHandler.build())

A FlxPoint that calls a function when set_x(), set_y() or set() is called. Used in FlxSpriteGroup. IMPORTANT: Calling set(x, y); is MUCH FASTER than setting x and y separately. Needs to be destroyed unlike simple FlxPoints!

Constructor

new(setXCallback:FlxPoint ‑> Void, ?setYCallback:FlxPoint ‑> Void, ?setXYCallback:FlxPoint ‑> Void)

If you only specify one callback function, then the remaining two will use the same.
