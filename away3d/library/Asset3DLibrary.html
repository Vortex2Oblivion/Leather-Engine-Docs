class Asset3DLibrary

package away3d.library

@:directlyUsed

Asset3DLibrary enforces a singleton pattern and is not intended to be instanced. It's purpose is to allow access to the default library bundle through a set of static shortcut methods. If you are interested in creating multiple library bundles, please use the getBundle() method.

@:value({ key : "default" })staticgetBundle(key:String = "default"):Asset3DLibraryBundle

Returns an Asset3DLibrary bundle instance. If no key is given, returns the default bundle (which is similar to using the Asset3DLibraryBundle as a singleton). To keep several separated library bundles, pass a string key to this method to define which bundle should be returned. This is referred to as using the Asset3DLibraryBundle as a multiton.

Parameters:

key

Defines which multiton instance should be returned.

Returns:

An instance of the asset library
