class SubMesh

package away3d.core.base

implements IRenderable

@:directlyUsed

SubMesh wraps a SubGeometry as a scene graph instantiation. A SubMesh is owned by a Mesh object.

See also:

  • away3d.core.base.SubGeometry

  • away3d.scenegraph.Mesh

Constructor

@:value({ material : null })new(subGeometry:ISubGeometry, parentMesh:Mesh, ?material:MaterialBase)

Creates a new SubMesh object

Parameters:

subGeometry

The SubGeometry object which provides the geometry data for this SubMesh.

parentMesh

The Mesh object to which this SubMesh belongs.

material

An optional material used to render this SubMesh.
